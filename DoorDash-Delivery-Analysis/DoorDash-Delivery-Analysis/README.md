# DoorDash Delivery Efficiency Analysis

![Dashboard Screenshot](dashboard-screenshot.png)

## ❌ The Problem
DoorDash operates across multiple hyper-local markets with highly erratic delivery patterns. I wanted to isolate the exact operational constraints influencing delivery speeds and order volumes to streamline regional fulfillment strategies.

## 🧰 Tools & Skills Used
*   **Tools:** Advanced Excel, PivotTables, PivotCharts, Conditional Formatting
*   **Skills:** Data Cleaning, Regional Segmentation, KPI Development, Dashboard Architecture

## 🔄 The Process
1. **Import & Clean:** Standardized variable types, removed duplicate entries, and verified data integrity inside `Beth Glasscock-Milestone-Doordash-Delivery-Analysis.xlsx`.
2. **Model Metrics:** Engineered calculated fields to track peak delivery times and baseline transit delays.
3. **Aggregate:** Built dynamic PivotTables to segment performance by major U.S. markets and consumer cuisine preferences.
4. **Visualize:** Constructed an interactive executive dashboard to turn raw logistics rows into rapid, scannable decisions.
